# **Capital Budgeting Financial Model using Discounted Cash Flow (DCF)**





## **Project Overview**



This project presents a comprehensive Capital Budgeting Financial Model developed in Microsoft Excel to evaluate the financial feasibility of a long-term investment project. The model applies industry-standard capital budgeting techniques to estimate project value and support investment decision-making.



The workbook is fully linked through formulas, allowing changes in assumptions to automatically update forecasts, financial statements, cash flows, and valuation outputs.







#### **Project Objective**



The objective of this project is to assess whether a proposed investment should be accepted by estimating its future cash flows and evaluating its financial performance using:



Discounted Cash Flow (DCF) Analysis

Net Present Value (NPV)

Internal Rate of Return (IRR)

Cost of Equity

Working Capital Analysis

Financing Analysis
